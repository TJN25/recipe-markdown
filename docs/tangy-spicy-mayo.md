# Tangy Spicy Mayo (2 serves)

??? note "Ingredients"

    ??? info "Base"
        - 4-5 tbsp mayo (Best Foods for creamier/richer, Kewpie for more umami/tang)

    ??? info "Spices"
        - 1 tsp Kashmiri chili powder (or 1-2 dried Kashmiri chilies, deseeded and crushed/ground fine)
        - ½ tsp smoked paprika
        - ½ tsp sumac
        - ¼ tsp garlic powder (or 1 small clove minced)

    ??? info "Tang"
        - 1-1½ tsp white vinegar (start with 1 tsp, add more to taste)

    ??? info "Optional"
        - Honey to taste if it needs balancing

??? note "Method"
    Mix everything together, let sit 10-15 minutes for flavors to meld. Taste and adjust vinegar/honey. If using dried chilies, consider toasting them briefly first for more depth, then grind.

??? note "Notes"
    - Kewpie is already tangier, so you might need less vinegar
    - If heat is too mild, add pinch of cayenne or crushed bird's eye
    - Makes enough for 2 generous sandwiches with a bit extra
